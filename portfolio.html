Project introduction

Software engineering bachelor, Third semester project a heterogenous system.
the project is a time logging system for a fictional company utilizing a three tier architecture.
The data tier was made using Entity Framework technology to provide acces to the relational database using object-mapping.
The logic tier is designed as an API written in Java that allows communication using HTTP requests.
And the presentation tier is build as a C# web application using the Razor pages framework.

Link to the github repository.





Project introduction

Software engineering bachelor, Second semester project a client-server system.
the project is a collaborative language teaching tool made with collaboration of a teacher in the sprogcenter midt horsens utilizing a client-server architecture.
The system was programmed in the Java programming language and using a relational database.
The server program communicates with the database using a singleton pattern connection and is the host for the classroom sessions.
The clients can connect to the classroom sessions created by the server program with the use of a socket protocol (RMI).

Link to the github repository.
